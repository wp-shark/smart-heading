== Description ==

**Smart Heading** is a Gutenberg block that simplifies the creation of visually appealing headers by combining titles and descriptions in one cohesive unit. Whether you need a simple page title or a complex section header, Smart Heading offers flexibility and control to design headers that stand out.

== Key Features of Smart Heading ==

**Title & Description Combination:** Merge titles and descriptions into a single block for more structured and visually balanced headings.

**Typography Control:** Full customization of fonts, sizes, weights, and colors to match your site's branding.
Responsive Design: Smart Heading is fully responsive, ensuring your headers look great on all devices.

**Gutenberg Integration:** Built to work seamlessly within the WordPress block editor, offering an intuitive drag-and-drop interface.

**Customizable Alignment:** Easily adjust text alignment (left, center, right) and add unique styling options to enhance the header's appearance.

This block is perfect for users who want more than just a simple heading element, offering the ability to add supporting descriptions without additional blocks or complexity.

== Frequently Asked Questions ==

=  How do I use Smart Heading in my content? =

After activating the Smart Heading plugin, open any page or post in the block editor, search for the Smart Heading block, and drag it into your content. You can then customize the title and description, along with typography and alignment settings, directly in the block editor.

= Can I customize the fonts and colors? =

Yes, Smart Heading offers full typography control, including font family, size, weight, color, and more, for both the title and description. You can easily adjust these settings in the block editor.

= How does this block improve my site's design? =

Smart Heading helps you create structured, stylish, and informative headers by combining a title and description in one block. This keeps your design clean and professional, while offering more control over how your headings appear on the page.
